### Project Overview
This project analyzes hotel booking patterns, revenue performance, and customer insights using Power BI and the Hotel Booking Demand dataset from Kaggle.

### Dashboard Pages
1. Booking Overview
2. Sales & Revenue Insights
3. Customer Insights

### Key Metrics
- Total Bookings
- ADR (Average Daily Rate)
- Revenue
- Cancellation Rate
- Total Guests
- Booking Trends

### Tools Used
- Power BI
- DAX
- Power Query
- Kaggle Dataset

### Files Included
- Hotel_Dashboard.pbix
- hotel_bookings.csv
- Dashboard Screenshots
- Project Documentation (README)
